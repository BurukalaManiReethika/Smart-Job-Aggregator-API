# 🚀 Smart Job Aggregator API

## 💡 Features
- Fetch job listings
- Filter by skill & location
- Remove duplicates
- Pagination support

## 🛠 Tech Stack
Python | Flask | REST API

## 🔗 Endpoints

GET /jobs  
GET /jobs?skill=python  
GET /jobs?location=remote  
GET /jobs?skill=python&limit=2  

## ▶️ Run

pip install -r requirements.txt  
python app.py
